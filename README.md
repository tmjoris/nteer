NTeer–Community Service Volunteer Monitoring System

NTeer is a web-based Community Service Volunteer Monitoring System designed to help community service students make informed decisions about where and when to volunteer.
The system provides real-time visibility of volunteer activity at registered community service sites, improving planning, efficiency, and overall user experience.

🎯 Objectives

Help students identify less crowded volunteering locations  
Improve coordination between volunteers and service sites  
Enhance time management for community service participants  
Provide real-time monitoring of volunteer distribution  

🚀 Features

📍 Real-Time Availability Tracking, View how many volunteers are present at each site  
🔍 Search & Filter Locations, Easily find volunteering opportunities by location  
👤 User Registration & Login, Secure authentication for students and administrators  
📊 Dashboard Analytics, Visual insights on volunteer trends and activity  
🏢 Site Management, Admins can add, update, and manage service locations  
⏰ Time Tracking, Monitor volunteer attendance and service hours  

🛠️ Technologies 

Frontend: HTML, CSS, JavaScript  
Backend: Node.js / Python  
Database: MongoDB / MySQL  
Tools: Visual Studio Code, Git, GitHub
